二叉树相关算法

发表于 2019-08-14 | 更新于: 2019-08-14 | 分类于 数据结构与算法 | 热度: ℃
字数统计: 778 | 阅读时长 ≈ 3

树是计算机领域经常用到的一种数据结构，因为树的定义是递归的，因此，树相关的算法也大都设计到递归求解。下面，我们针对树的常见算法进行一一解析。

根据二叉树的前序遍历序列与中序遍历序列构建二叉树

算法描述

给出一颗二叉树的前序遍历与中序遍历序列，重构出这颗二叉树。

用例示例：

preorder = [3,9,20,15,7]
inorder = [9,3,15,20,7]

返回二叉树:

3
/ \
9 20
/ \
15 7

算法分析

前序遍历的顺序是root->left->right，根据前序遍历可以确定根节点位置，但无法知道左子树与右子树的界限。而中序遍历的顺序left->root->right可根据根节点的位置确定左子树与右子树的界限。两者结合便可确定唯一的一颗二叉树。

实现代码

‘’’java
/*
\ 树节点定义如下：.
* public class TreeNode {
* int val;
* TreeNode left;
* TreeNode right;
* TreeNode(int x) { val = x; }
* }
*/
class Solution {
public TreeNode buildTree(int[] preorder, int[] inorder) {
if(preOrder==null||inOrder==null) return null;
return buildTree(preorder,inorder,0,0,inorder.length-1);
}

// 采用递归的方式构建二叉树。递归构建二叉树过程中我们需要知道先序序列中当前根节点
// 的位置preIndex,中序遍历中当前子树所有节点的位置区间
private TreeNode buildTree(int[] preorder, int[] inorder, int preIndex,inStart,inEnd){
    // 递归出口。当根节点位置大于先序序列长度或中序序列起始位置大于结束位置时，
    // 说明当前节点左子树或右子树为空，结束递归。
    if(preIndex>preOrder.length-1||inStart>InEnd){
        return null;
    }
    // 按先序序列顺序创建根节点及子树根节点
    TreeNode node = new TreeNode(preorder[preIndex]);
    // 保存根节点在中序序列中的位置
    int currIndex=0;
    // 找到根节点在中序序列中的位置
    for(int i=0;i<inorder.length;i++){
        if(inorder[i]==preorder[preIndex]){
            currIndex=i;
        }
    }
    // 递归创建左子树。根据先序遍历访问节点顺序root|left|right,可知左子树根节
    // 点位于先序序列当前根节点位置右边第一个位置，即preIndex+1,整个左子树在中
    // 序序列中的位置区间为当前中序序列起始位置到当前根节点在中序序列中的位置左边
    // 第一个位置,即[inStart,currIndex-1]。
    node.left = buildTree(preorder,inorder,preIndex+1,inStart,currIndex-1);
    // 递归创建右子树。根据先序遍历访问节点顺序root|left|right,可知右子树根节
    // 点位于先序序列当前根节点位置加上左子树节点个数再加一，左子树节点个数又等于
    // 中序序列中根节点的位置减去当前起始位置即currIndex-inStart,因此右子树根
    // 节点位置为preIndex+currIndex-inStart+1,整个右子树在中序序列中的位置
    // 区间为当前根节点在中序序列中的位置加一到当前中序序列结束位置即
    // [currIndex+1,inEnd]。
    node.right = buildTree(preorder,inorder,preIndex+currIndex-inStart+1,currIndex+1,inEnd);
    return node;
}

}
